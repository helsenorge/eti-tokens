# Design tokens for ETI

Adapted from https://github.com/digdir/designsystemet

## For å generere tokens fra design-token mappe

 `npx @digdir/designsystemet@latest generate-config-from-tokens --dir design-tokens`

## For å generere tokens fra config-fil

`npx @digdir/designsystemet@latest tokens create --config designsystemet.config.json`

## For å generere CSS-variabler fra tokens

`npx @digdir/designsystemet@latest tokens build --config designsystemet.config.json`
